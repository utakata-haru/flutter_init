# ボイスメモアプリ – アプリケーション仕様書（最終版 v1.0）

最終更新: 2025-08-22
作成者: App Builder（AI）

## 1. コンセプト / 目的
素早く「録る・見返す・見つける」体験に特化した、個人利用のボイスメモアプリ。外部同期に依存せず端末内で完結し、録音から検索・再生・整理までを最短動線で提供する。

## 2. 対応プラットフォーム（MVP）
- iOS（優先）
- Android（限定対応: 録音/再生/一覧を中心に、iOSと同等UXは後追い）

## 3. MVP 機能
- 録音/停止/保存、バックグラウンド録音（OS許可範囲）
- 音声再生（シーク/再生速度/スキップ）
- メタ情報編集（タイトル、スター、メモ）
- タグ付け（複数タグ、簡易管理）
- 検索（テキスト全文・タグ・期間）
- 文字起こし（手動トリガーのみ、完了後に全文検索対象へ反映）
- 削除（ゴミ箱→復元/完全削除）
- 設定（録音品質、既定保存先、プライバシー）

## 4. 画面構成（MVP）
- ホーム: 録音UI + 一覧タブ/検索タブ
- 詳細: 再生・タイトル/タグ/メモ編集、文字起こしの実行/結果表示
- 検索: キーワード/タグ/期間のフィルタ
- 設定: 録音品質、プライバシー方針表示
- ゴミ箱: 30日保管、個別/一括復元・削除

## 5. 主要フロー（MVP）
1) 録音開始 → 停止 → 保存（タイトル自動案: 日時）
2) 一覧から選択 → 詳細で再生/編集
3) 詳細で「文字起こしを実行」→ アダプタ経由で処理 → テキスト保存
4) 検索（テキスト/タグ/期間）→ 結果から詳細へ
5) 削除 → ゴミ箱 → 復元 or 完全削除

## 6. データモデル（論理）
- VoiceMemo
  - id (string/uuid)
  - title (string)
  - tags (List<string>)
  - isStarred (bool)
  - note (string?)
  - durationMs (int)
  - audioPath (string)
  - createdAt (DateTime)
  - updatedAt (DateTime)
  - isTrashed (bool)
  - transcriptionText (string?)
  - transcriptionStatus (enum: none/pending/success/failed)
- Tag
  - id (string)
  - name (string, unique)

## 7. 音声仕様（MVP）
- フォーマット: AAC (m4a)
- モノラル / 44.1kHz / 96–128kbps（設定で変更可・デフォルト中）
- ファイル保存: アプリ専用ディレクトリ（端末内）

## 8. 文字起こし仕様（MVP）
- 実行: 手動のみ（自動は将来）
- 実装: Infrastructure 層の transcription_adapter 経由で実行
- 状態: none → pending → success/failed（履歴は保持しない）
- 検索: success 時に全文検索対象へ反映

## 9. 検索/整理
- フリーテキスト（タイトル・メモ・文字起こし）
- 絞り込み（タグ/期間/スター）
- 並び替え（作成日時・更新日時・タイトル）

## 10. 削除/安全性
- ソフトデリート: isTrashed=true でゴミ箱へ
- ゴミ箱: 30日保持、復元/完全削除
- 永久削除: 音声ファイル + メタを物理削除

## 11. 再生体験
- 再生速度（0.75x/1.0x/1.25x/1.5x/2.0x）
- 15秒戻る/30秒送る（カスタムは将来）
- ロック画面/バックグラウンドの基本制御（OS標準範囲）

## 12. 非機能要件
- パフォーマンス: 起動<1.5s目標、録音開始遅延最小化
- 安定性: 録音中断/復帰の堅牢化、クラッシュ0.5%未満
- プライバシー: 端末内完結、解析/広告SDKなし
- アクセシビリティ: VoiceOver/フォントサイズ対応

## 13. Android要件（MVP範囲）
- 録音/再生/一覧/詳細/検索の基本動作
- 一部UI/権限ダイアログはOS差異に合わせる

## 14. 受け入れ基準（抜粋）
- 録音→保存→一覧→詳細→再生が途切れなく可能
- 検索（テキスト/タグ/期間）が期待どおりに絞り込める
- 文字起こしを手動実行し、成功時に結果が保存/検索可能
- ゴミ箱で復元/完全削除が機能

## 15. 将来機能（v1.1+）
- 自動文字起こし、クラウド同期、共有、リマインダ、AI要約

## 16. アウトオブスコープ（MVP）
- 外部クラウド連携、複数端末同期、チーム共有、Webクライアント
